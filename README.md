# DeepNP
## Installation
Installer les dépendances sur un environnement python
```
$ pip3 install -r requirements.txt
```
Exécuter les commandes suivantes dans un terminal
```
$ cd datasets/
$ ./get_datasets.sh
$ cd ../utils/cython/
$ ./build_cython.sh
```
## Exécuter
Ouvrir les notebooks avec JupyterLab ou un éditeur de votre choix
```
$ jupyter lab
```
